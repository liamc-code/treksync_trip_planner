
# TrekSync Trip Planner

TrekSync Trip Planner is trip Organizer and planning App built for trip creation and management. It allows users to add, edit, and view trips, ensuring smooth and organized travel planning. Ensure dependencies and plugin versions are up to date before running.

## Features

- User-friendly interface for trip planning
- Add, edit, and delete trip details
- View a comprehensive list of all planned trips
- Supports multiple customer types (Family, Group, Individual)
- Validation for inputs (e.g., email format, phone number format)
- Dynamic total price calculation for all trips
- Responsive design for mobile devices

## Requirements

- Android device or emulator
- Android Studio (for development purposes)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/liamc-code/treksync_trip_planner.git
   ```

2. Open the project in Android Studio.

3. Build and run the project on your preferred emulator or connected Android device.

## Usage

1. **Add a New Trip**:
   - Select the "Add Trip" button.
   - Fill in the trip details, including customer type, destination, contact information, and any relevant specifics like family insurance, policy number, or home address.
   - Click "Save Trip" to add the trip to the list.

2. **Edit a Trip**:
   - From the trip list, click the edit icon next to the trip you want to modify.
   - Update the trip details as necessary.
   - Save the changes by clicking "Save Trip."

3. **Delete a Trip**:
   - In the trip list, click the delete icon next to the trip you want to remove.
   - Confirm the deletion.

4. **View Trip List**:
   - Navigate to the trip list screen to see all added trips.
   - The total price for all trips is dynamically displayed at the top.

5. **Input Validation**:
   - Ensure correct formatting for email addresses and phone numbers.
   - Invalid inputs will display appropriate error messages.
